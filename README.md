# 🛠️ Tech stack

- react 19
- next 16.2.6 https://nextjs.org/docs
- sanity https://www.sanity.io
- next-intl

# 🚀 Run

```sh
npm i
npm run dev
```

Configure and connect sanity, or mock data will be used.

```sh
NEXT_PUBLIC_SANITY_PROJECT_ID=
NEXT_PUBLIC_SANITY_API_VERSION=
NEXT_PUBLIC_SANITY_DATASET=
```
